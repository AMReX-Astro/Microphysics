# `he-burn`

This contains various networks that are intended for burning He (and
perhaps H).  The network names contain the number of nuclei, and
several different letters:

* `a` : the (α, p)(p, γ) approximation is used

* `m` : one or more modified rates are used to connect odd Z nuclei to even Z

* `n` : the (n, γ)(n, γ) approximation is used

* `p` : NSE protons are used at high A
